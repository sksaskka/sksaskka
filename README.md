# Практическое задание

## Вариант - 4
Программа создаёт 3 потока, каждый считает свой диапазон чисел; общий результат собирается после Join.

## Как это работает:
Программа считает сумму чисел от 1 до 300, разделяя работу на три равных части:
* **Поток 1:** считает сумму чисел от 1 до 100
* **Поток 2:** считает сумму чисел от 101 до 200
* **Поток 3:** считает сумму чисел от 201 до 300

Главный поток программы запускает их одновременно, а затем с помощью метода `Join()` ждёт, пока каждый поток закончит вычисления, чтобы сложить промежуточные результаты в общую сумму.

Пример работы программы:

# Первый поток: 5050
# Второй поток: 15050
# Третий поток: 25050
# Общая сумма: 45150
